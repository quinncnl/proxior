Proxior

A lightweight proxy manager proxy server

Download this project as a .zip file Download this project as a tar.gz file

What is it?

Proxior is a proxy manager server, and can be seen as a proxy router. It works like SwitchySharp and PAC(Proxy auto-config). However, SwitchySharp works only with Chrome Desktop while PAC requires a cumbersome Javascript engine. These limits restrict their usage on mobile devices. Proxior comes out as a sollution.

Proxior是一个管理代理服务器的HTTP(S)代理服务器，可被视为一个HTTP层的路由 器。它的工作方式与 SwitchySharp 和 PAC(代理自动配置)相似。但 SwitchySharp 只能在Chrome桌面版使用，PAC需要Javascript解释器。这 些局限使其无法在移动设备上使用。Proxior成为一种解决方法。

Features

  • Single thread; consumes super little memory and CPU.
  • Manage multiple proxy profiles and URL lists.
  • Supports HTTP and Socks5 proxy.
  • Supports Ubuntu, Mac and possibly other *nix distributions.

    • Remote Control

    You can manage lists and check log online.

    安装与使用

    需要 libevent2
    在Ubuntu 12.04和OSX 10.8测试通过
    在Ubuntu下， 进入程序目录 
    #安装libevent
sudo apt-get install libevent-dev
make
#测试运行
./run
#安装
make install
#反安装
make uninstall

    配置

    用./run运行配置在代码目录，安装后用proxior运行时配置目录在/etc/proxior

    proxy PROXYNAME PROXYIP:PROXYPORT

    添加一个名为 PROXYNAME 的HTTP代理， 代理地址为 PROXYIP:PROXYPORT

    socks5 PROXYNAME PROXYIP:PROXYPORT

    添加一个名为 PROXYNAME 的SOCKS5代理(可用ssh -D取得，不支持身份认证)， 代理地址为 PROXYIP:PROXYPORT

    acl PROXYNAME LIST_FILENAME

    添加一个名为 LIST_FILENAME 的规则列表，它与proxy.conf同目录，满足这个 规则表的请求用PROXYNAME这个代理访问
    优先级:若一个URL满足多个列表，则使用第一个满足的列表

    acl-default PROXYNAME

    不在列表的请求用 PROXYNAME 访问

    acl-try PROXYNAME

    如果连接失败，如重置等，尝试使用 PROXYNAME 连接

    timeout TIME

    与服务器连接的超时时间

    listen HOST:PORT

    在 HOST:PORT 建立HTTP代理

    示例配置

    proxy SSH 192.168.0.3:8123
proxy GAE 192.168.0.3:8087
proxy Squid 192.168.0.3:3128
socks5 dxt localhost:7070

#此处的优先级: directlist > gaelist > dxtlist > sshlist
acl direct directlist
acl GAE gaelist
acl dxt dxtlist
acl SSH sshlist

acl-default direct
acl-try SSH

timeout 30
listen 0.0.0.0:9999

    Tips

    1. Ubuntu下使用Upstart建立SSH隧道，可设置为连接中断或程序终止后自动重连
    2. 先用./run看看程序的工作情况，再用proxior -d以守护进程运行
    3. 几乎所有常用操作系统(Windows, OSX, Ubuntu, iOS, Android等)都可以 使用HTTP代理
    4. 优先级很有用
