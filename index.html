What is it?

Proxior is a proxy manager server, and can be seen as a proxy router. It works like SwitchySharp and PAC(Proxy auto-config). However, SwitchySharp works only with Chrome Desktop and PAC requires Javascript engine. These limits disable their usage on mobile devices. Proxior come out as a sollution.

Proxior是一个管理代理的HTTP代理服务器，可被视为一个HTTP层的代理路由 器。它的工作原理与 SwitchySharp 和 PAC(代理自动配置)相似。但 SwitchySharp 只能在Chrome桌面版使用，PAC需要Javascript解释器。这 些局限使其无法在移动设备上使用。Proxior成为一种解决方法。

Features

  • Single thread; consumes supper little memory and CPU.
  • Manage multiple proxy profiles and URL lists.
  • Supports Ubuntu, Mac and possibly other *nix distributions.
